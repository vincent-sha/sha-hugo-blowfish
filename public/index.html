OpenClaw 再曝严重安全隐患

2026-04-07·1 分钟
科技资讯 OpenClaw AI 安全漏洞
OpenClaw 再曝严重安全隐患 # 快速导览 # OpenClaw 是一款自 2025 年底走红的“agentic” AI 工具（Github 项目：https://github.com/openclaw/openclaw），其设计目标是代替或模仿用户执行多种任务——管理文件、与各类通信工具和服务交互、调用已登录的会话与凭据等。为了实现这些能力，OpenClaw 需要对大量本地与联网资源拥有广泛访问权限，这也带来了严重的安全风险。 2026 年 4 月披露并修复的高危漏洞 CVE-2026-33579 可在没有额外交互的情况下，将拥有最低“operator.pairing”权限的设备悄然升级为“operator.admin”，从而完全接管 OpenClaw 实例并访问其连接的所有数据源与凭据。 漏洞影响 # 漏洞评级：根据不同评分矩阵，严重度在 8.1 到 9.8（满分 10）。 本质：任何持有配对（pairing）权限的设备，如果能发起“配对批准”请求，即可被静默批准为拥有管理员权限的设备（operator.admin）。 后果：一旦获得管理员权限，攻击者可以读取所有连接的数据源、窃取存储在 agent 技能环境（skill environment）中的凭据、执行任意工具调用，并横向转移到其他服务——实质上是完整的实例接管（full instance takeover）。 研究总结：在许多企业部署场景中，这次漏洞的影响等同于“不需要二次漏洞、不需要额外交互、无需用户干预“即可完全接管。

Google 推出的实时语音翻译功能，国区如何用？ # 快速导览 # Google Translate 的实时语音翻译（Live Translate）功能现已登陆 iOS。该功能由 Google 的大模型 Gemini 驱动，支持将对话或影音中的语音实时翻译并通过耳机播报，让跨语言交流更加顺畅自然。 功能亮点 # Gemini 驱动：在翻译时尽量保留说话者的“语气、重音与语速节奏”，以获得更自然的听感。 耳机实时收听：将翻译通过蓝牙耳机直接播出，适合与人面对面交谈或观看外语影视时同步收听翻译。 无需耳机也可工作：即便不接耳机，Live Translate 仍能进行实时翻译。 支持语言：覆盖 70 多种语言（如阿拉伯语、中文、英语、法语、德语、印地语、意大利语、葡萄牙语、西班牙语等）。 快速上手 # 在 App Store 下载并打开 Google Translate（或确保应用为最新版本）。 在首页选择“源语言”和“目标语言”。 连接蓝牙耳机（或使用设备扬声器）。 点击首页左下角的 “Live Translate”（实时翻译）按钮开始。 (示意：应用内的 Live Translate 按钮)

喜欢胶片摄影吗？值得入手的胶片相机推荐 # 为什么拍胶片？ # 胶片摄影与数码最大不同在于节奏与结果的不可即刻校正——你需要等待冲洗与扫描，这个过程反而能让摄影变得更有仪式感和思考度。近年社交平台对“film look”的偏爱、名人频繁晒胶片照片也推动了胶片文化回潮。无论是想挑战自我、追求独特影调，还是单纯喜欢复古质感，胶片都是值得尝试的媒介。 如何选择 # 选择相机前先考虑：预算、是否想要自动化（测光/对焦）、是否希望可换镜头、以及你想拍的片幅（35mm vs 中画幅）。下面按常见类型给出简介与推荐。 旁轴 / 口袋傻瓜机（Point-and-shoot） # 特点：镜头固定、通常带自动对焦与自动曝光，最省心的入门选择。 优点：轻便、上手快、适合街拍与日常记录。 推荐机型：Canon Prima 系列（入门价位约 $50–$130）; Konica Hexar AF（高端、约 $800–$1,000，镜头素质好）。 复古单反（Vintage SLR） # 特点：很多老相机为全机械或半机械设计，需手动对焦与拨环设置快门/光圈。 优点：便宜、镜头选择多、耐用；能学到传统测光与曝光控制技能。 代表机型：Canon AE-1、Nikon F、Pentax K1000（Pentax K1000 常见且价格低廉）。 现代单反（Later SLR）——最适合初学者入门 # 特点：更现代的自动化（自动曝光、自动对焦、可换镜头），很多与数码时代的镜头接口相同（如 Canon EOS 系列的 EF 卡口）。 优点：使用体验接近数码相机，上手与控制都更简单，二手市场充足且便宜。 代表机型：Canon EOS 1000F（便宜且常见，约 $20–$50）。 中画幅与大画幅（进阶/爱好者） # 特点：使用 120（或更大）底片，单次拍摄能记录更多细节与更优的景深表现。 缺点：机身与胶片、冲扫成本高；流程与操作更复杂。 代表机型：Hasselblad 500C（经典但昂贵）、Mamiya 6、Mamiya 645（性价比更高的中画幅）。 推荐机型速览 # 相机类型 推荐机型 价格区间（二手） 适合人群 口袋傻瓜机 Canon Prima（入门） / Konica Hexar AF（高端） $50–$1,000 想随手拍、怕复杂设置的初学者 复古 SLR Pentax K1000 / Canon AE-1 / Nikon F $10–$200 想学手动曝光、预算有限的入门者 现代 SLR Canon EOS 1000F 等 $20–$100 想要自动化支持、并能使用现代镜头的人 中画幅 Hasselblad 500C / Mamiya 645 / Mamiya 6 几百到数千美元 追求更高画质与更大底片的进阶用户 注：二手价格受成色、是否附镜头、是否需修理等影响较大，表格仅作大致参考。
